# Target Model
This is the Target model extension. It provides a model of Target or TrackedTarget objects. A TrackedTarget or Target 
generally refers to an object which is detected by a detection or tracking system such as RADAR.

## Getting Started


### Prerequisites
In order to install the model extension, the parent eMir-OpenSource project needs to be installed. This can be done by executing
following command:

    mvn clean install

### Installation
The Target model extension is either installed when executing mvn clean on the parent eMir-OpenSource or can be installed manually
by executing the following command in this directory:

    mvn clean install
